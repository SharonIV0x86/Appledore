Appledore

A simple and efficient graph library in modern C++. Supports directed/undirected graphs with customizable edge type and vertices, designed for performance and ease of use.

C++20 Header Only MIT License

Features

Adjacency Matrix

Efficient matrix-based representation for dense graphs.

Adjacency List

Optimized list-based representation for sparse graphs.

Custom Types

Full support for custom vertex and edge types (classes/structs).

Easy Querying

Simple API to query graph structure and properties.

Upcoming Features: Support for graph degrees, advanced algorithms (Dijkstra, BFS, DFS), and analysis utilities.

Installation

Appledore is easy to integrate. Follow these steps to get started:

1

Clone the repository

git clone https://github.com/SharonIV0X86/Appledore
2

Include the header

Include the GraphMatrix.hpp header in your project files.

#include "../include/GraphMatrix.hpp"

Directed Graph Example

Implementation of a Graph relationship between Airports and Distances.

main.cpp 
#include <iostream>
#include "../include/GraphMatrix.hpp"

using namespace Appledore;

class Airport : public GraphVertex {
public:
    std::string name;
    Airport(std::string name) : name(name) {};
    friend std::ostream &operator<<(std::ostream &os, const Airport &airport) {
        os << airport.name;
        return os;
    }
};

class Flight {
public:
    int distance;
    Flight(int dis) : distance(dis) {};
    friend std::ostream &operator<<(std::ostream &os, const Flight &flight) {
        os << flight.distance << " miles";
        return os;
    }
};

int main() {
    GraphMatrix<Airport, Flight, DirectedG> AirportsGraph;
    // ... Additional setup code ...
}

Undirected Graph Example

Using UndirectedG for bidirectional relationships.

GraphMatrix<Airport, Flight, UndirectedG> AirportGraph;
AirportGraph.addVertex(LAX, JFK, DEN, ATL);
AirportGraph.addEdge(LAX, DEN, flight1);

Mixed Graph Example

Using MixedGraphMatrix to combine different transportation modes.

Appledore::MixedGraphMatrix<State, Connection> americaNetwork;
americaNetwork.addEdge(california, texas, roadConnection, false); // false = undirected
americaNetwork.addEdge(texas, florida, airConnection, true);      // true = directed
